Readme to dont's like road Dev

🗂️ โปรเจคแนะนำ แยกตามสาย

🎨 Frontend — Personal Finance Dashboard

ติดตามรายรับ-รายจ่ายตัวเอง แสดงกราฟ สรุปรายเดือน

ทำไมถึงดี: เห็นผลทันที สวยงามได้ ฝึก HTML/CSS/JS หรือ React
ได้เรียนรู้: Layout, State management, Chart library
ความรู้สึกถ้าชอบสาย Frontend: สนุกกับการจัด UI ให้สวย อยากให้คนใช้แล้วรู้สึกดี

⚙️ Backend — Line Notify Bot แจ้งเตือนส่วนตัว

ส่ง notification เข้า Line ตัวเอง เช่น แจ้งอากาศวันนี้, reminder ดื่มน้ำ, ราคาหุ้น

ทำไมถึงดี: ใช้จริงทุกวัน ฝึก API, HTTP Request, Cron job
ได้เรียนรู้: REST API, Server logic, การเชื่อมต่อ 3rd party
ความรู้สึกถ้าชอบสาย Backend: ชอบให้ระบบทำงานเองโดยไม่ต้องเปิดหน้าจอ

📊 Data / AI — วิเคราะห์พฤติกรรมการใช้จ่ายตัวเอง

นำ statement จากธนาคารมาวิเคราะห์ หาว่าจ่ายเยอะสุดหมวดไหน

ทำไมถึงดี: Data จริงของตัวเอง เห็นผลชัดเจน
ได้เรียนรู้: Python, Pandas, Data visualization, pattern recognition
ความรู้สึกถ้าชอบสาย Data: ชอบหาคำตอบจากตัวเลข สนใจ "ทำไม" มากกว่า "ทำอย่างไร"

📱 Mobile — Habit Tracker App

แอปติดตาม habit รายวัน เช่น ออกกำลังกาย อ่านหนังสือ กด check ได้

ทำไมถึงดี: ใช้บนมือถือจริง เห็น progress ตัวเอง
ได้เรียนรู้: React Native หรือ Flutter, Local storage, UI/UX mobile
ความรู้สึกถ้าชอบสาย Mobile: ชอบทำของที่อยู่ในมือทุกวัน ใส่ใจ user experience

🌐 Full Stack — ระบบจดบันทึก + แชร์ลิงก์ได้

สร้าง note ส่วนตัว แก้ไขได้ แชร์เป็น public link ได้ เหมือน Notion เล็กๆ

ทำไมถึงดี: ครอบคลุมทุกส่วน ตั้งแต่ UI → API → Database
ได้เรียนรู้: Frontend + Backend + DB ครบในโปรเจคเดียว
ความรู้สึกถ้าชอบ Full Stack: ชอบเห็นภาพรวมทั้งหมด ไม่ชอบติดอยู่แค่ส่วนเดียว
